    Spring框架文档 #

    Version 5.2.2.RELEASE

    新内容、升级说明、支持的版本和其他主题独立于发布节奏，在项目的Github Wiki上进行外部维护。

    概述 #

    历史，设计理念，反馈，入门。

    核心技术 #

    IoC容器、事件、资源、i18n、验证、数据绑定、类型转换、SpEL、AOP。

    测试 #

    Mock对象，TestContext框架，Spring MVC测试，WebTestClient。

    数据访问 #

    事务、DAO支持、JDBC、O/R映射、XML编组。

    Web Servlet #

    Spring MVC，WebSocket，SockJS，STOMP消息。

    响应式Web #

    Spring WebFlux，WebClient，WebSocket。

    集成 #

    远程处理、JMS、JCA、JMX、电子邮件、任务、调度、缓存。

    语言 #

    Kotlin，Groovy，动态语言。
